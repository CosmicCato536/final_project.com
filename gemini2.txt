Для TF2 (Team Fortress 2) такая структура подходит идеально, потому что у одного класса (например, Солдата) может быть много вариантов оружия, а один и тот же дробовик может подходить разным классам.
Вот пример того, как логически и программно наполнить базу данными по TF2.
## 1. Подготовка справочников
Сначала нужно наполнить таблицы «справочники» (оружие и способности), чтобы у них появились ID.

# Добавляем оружие (оно общее для всей игры)weapons_data = [
    ("Ракетомет", "Прямой урон по площади"),
    ("Дробовик", "Вторичное оружие для многих классов"),
    ("Священная камбала", "Рыба для Разведчика")
]
cursor.executemany("INSERT INTO weapons (name, specifications) VALUES (?, ?)", weapons_data)
# Добавляем способностиabilities_data = [
    ("Двойной прыжок",),
    ("Невидимость",),
    ("Лечение",)
]
cursor.executemany("INSERT INTO abilities (text) VALUES (?, ?)", abilities_data)
conn.commit()

## 2. Создание персонажа со связями
Допустим, мы создаем Разведчика (Scout). У него есть «Двойной прыжок» (способность №1) и он может носить «Дробовик» (оружие №2) и «Рыбу» (оружие №3).

# 1. Создаем самого Скаута
cursor.execute("INSERT INTO character (name, hp, img_path) VALUES (?, ?, ?)", 
               ("Разведчик", 125, "scout.png"))scout_id = cursor.lastrowid # Получаем его ID (например, 1)
# 2. Привязываем ему способность (Двойной прыжок имеет ID 1)
cursor.execute("INSERT INTO character_abilities (character_id, ability_id) VALUES (?, ?)", 
               (scout_id, 1))
# 3. Привязываем ему сразу два оружия (Дробовик ID 2 и Рыба ID 3)scout_arsenal = [2, 3]for weapon_id in scout_arsenal:
    cursor.execute("INSERT INTO character_weapons (character_id, weapon_id) VALUES (?, ?)", 
                   (scout_id, weapon_id))

conn.commit()

## Как это выглядит в DB Browser
Вам не нужно искать ID оружия в коде, вы просто смотрите их в таблицах weapons и abilities.

* В таблице character у вас будет одна строка: 1 | Разведчик | 125.
* В таблице character_weapons будет две строки:
* 1 | 2 (Скаут владеет Дробовиком)
   * 1 | 3 (Скаут владеет Рыбой)

## Почему это круто для TF2?
Если вы решите добавить Инженера, вы просто:

   1. Создадите его в character.
   2. В таблицу character_weapons добавите строку с его ID и тем же самым ID Дробовика (2).
   Вам не нужно создавать дробовик заново — он один в базе, но принадлежит многим персонажам.
